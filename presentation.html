安裝與使用

如何開始使用

安裝步驟

  1. 下載擴充功能原始碼
  2. 開啟 Chrome 擴充功能頁面 (chrome://extensions)
  3. 開啟「開發者模式」
  4. 點擊「載入未封裝項目」
  5. 選擇資料夾
  6. 設定 API Key 即可使用

使用須知

  • 需要給予麥克風權限
  • 自備 OpenAI/Google API Key
  • 建議使用付費方案避免限制
  • 僅支援桌面版 Chrome/Edge
